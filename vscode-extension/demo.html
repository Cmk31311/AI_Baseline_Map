  1. Open VS Code
  2. Open the vscode-extension folder

üîß Troubleshooting

  • F5 not working: Make sure you're in the extension folder and have the .vscode folder with launch.json
  • No hover information: Check that you're hovering over supported features in the right file types
  • Extension not loading: Run npm run compile to rebuild the extension
